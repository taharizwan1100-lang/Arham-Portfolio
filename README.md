# Arham Ali - Portfolio

![Portfolio Screenshot](https://images.unsplash.com/photo-1519389950473-47ba0277781c?auto=format&fit=crop&w=1600&q=80)

## Description

This is a **personal portfolio website** for Arham Ali, a Web Development student (ID: 640330) under the guidance of Sir Umar.  
The portfolio showcases personal information, skills, education, and career goals in a **modern, responsive design** with smooth scroll animations.

The design is built using **Tailwind CSS**, with custom colors, gradients, and shadow effects defined in `tailwind.config.js`. Scroll animations are powered by **AOS (Animate On Scroll)**.



## Features

- Responsive and mobile-friendly layout  
- Modern hero section with background image  
- About Me section with personal details  
- Skills section showcasing web development skills  
- Education section with study details  
- Career Goals section  
- Contact section with email link  
- Smooth scroll animations for all sections  
- Clean gradient header and footer theme  

---

## Technologies Used

- HTML5  
- Tailwind CSS  
- JavaScript (for AOS animations)  
- AOS (Animate On Scroll library)  
- Unsplash images for background and placeholders
